Salute by mattfeel

The objective of this project is to create a web-based medical management portal in which a patient retains control over their own medical records, yet provides easy access to any authorized healthcare provider. This project is envisioned as part social network, part medical information database that will allow for patient authorized “sharing” of relevant medical records to various doctors and specialists.
